## About
* Mine은 경매 서비스를 제공하기 위한 프로젝트입니다.
* C2C 거래 형태를 지원합니다.
* 상품을 구매하기 위해서는 입찰을 하고 낙찰되어야 합니다. 만약 판매자가 즉시 구매가를 설정했다면 즉시 구매도 가능합니다.
* 자동 입찰을 설정할 수 있습니다. 지불할 의사가 있는 최대 금액을 입력하면 사용자를 대신하여 증분 입찰가로 응찰합니다.
